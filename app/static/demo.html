LMT Defence – Intercept Demo

This demo simulates the radar system:

  • Calls /intercept once at start to get static intercept point C
  • Calls /radar/next every second to get updated threat position B(t) from API
  • Calculates interceptor position I(t) in HTML, moving from base A toward C
B(t): Threat lat/lon updated from API | C: Static intercept point | I(t): Interceptor managed in HTML
